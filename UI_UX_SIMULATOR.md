# UI/UX SPECIFICATION: COMMAND CENTER DASHBOARD

## Visual Style
- **Concept**: Bảng điều khiển Quốc gia (National Strategic Command Dashboard).
- **Theme**: Sang trọng, Đỏ cờ (`#C8102E`), Vàng kim (`#FFCD00`), Nền tối hoặc Nền kem giấy cũ (`#F4EFEB`).
- **Fonts**: Be Vietnam Pro, Montserrat.

## Host Interface (Projector Screen)
1. **Top Bar**: 
   - Header: "MÔ PHỎNG QUYẾT SÁCH ĐỔI MỚI (1986 - 2021)"
   - Live Status Gauge: 
     * 📈 GDP/Kinh tế (Bar chart)
     * ❤️ Đời sống Nhân dân (Bar chart)
     * 🌐 Vị thế Quốc tế (Bar chart)
2. **Main Stage**:
   - Timeline Marker (Highlighting current Year, e.g., 1986).
   - Crisis Statement Box (Hiển thị vấn đề lịch sử).
   - Live Voting Counter (% lựa chọn của cả lớp nhảy realtime theo dạng biểu đồ tròn/cột).
3. **Outcome Screen**:
   - Sound effect / Animation: Chỉ số nhảy số (+25 GDP, -700% Lạm phát).
   - Unlocked Milestone Badge (ví dụ: "Đã mở khóa: Top 3 Nước Xuất Khẩu Gạo").

## Player Interface (Mobile)
1. **Header**: Hiển thị tên người chơi + Số điểm Đóng góp (Contribution Score).
2. **Main Action Area**:
   - Các nút chọn Quyết sách A / B với mô tả ngắn gọn, dễ chạm.
   - Nhận phản hồi ngay lập tức: "Lựa chọn của bạn đã được ghi nhận vào Hội đồng!"
